Araya Client WebView 0.3.1
---------------------------------
1) Залейте на GitHub папку araya-client-webview-0.3.1 целиком (в корень репо).
2) Repo → Settings → Secrets and variables → Actions:
   - GOOGLE_CLIENT_ID
   - GOOGLE_CLIENT_SECRET
3) Actions → Build Araya Client WebView (Windows) → Run workflow.
4) Скачайте артефакт araya-client-webview-dist.zip → внутри portable .exe.

Локально: создайте .env на основе .env.example, затем `npm i` и `npm start`.
